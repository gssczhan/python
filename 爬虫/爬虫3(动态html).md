## Selenium + PhantomJS
- Selenium: web自动化测试工具
    - 自动加载页面
    - 获取数据
    - 截屏
    - 安装: pip install selenium==2.48.0
    - 官网: http://selenium-python.readthedocs.io/index.html
- PhantomJs
    - 基于Webkit的无界面的浏览器
    - 官网: http://phantomjs.org/download.html
- Selenium库有一个WebDriver的API
- WebDriver可以跟页面上的元素进行各种交互, 用它可以来进行爬取
